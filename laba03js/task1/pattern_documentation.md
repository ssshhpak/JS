# Input Validation Utility Pattern 

## Pattern Name
Input Validation Utility

## Description 
Provides reusable validation logic for user input.

## Context Outline 
Used in web forms, APIs, and authentication systems.

## Problem Statement
Invalid user input causes system failures.

## Solution
1. Accept input 
2. Validate format
3. Return result
4. Handle errors

## Design 
The pattern uses standaline utility functions.

## Implementation
Implemented as reusable eS6 functions.

## Examples
See pattern_example.js

## Consequences
### Benefits
- Reusability
-Reliability

### Trade-offs
- Extra validation logic
